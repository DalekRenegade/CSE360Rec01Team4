▣ What is a binary number?
► A binary number is a sequence of binary digits(1 or 0). Each digit is referred to as a bit.
     The binary system is used internally by almost all modern computers and computer-based devices.

        Here are some examples of binary numbers and their corresponding decimal values.
                0000 0001 = '1'
                0000 0010 = '2'
                0000 1111 = '15'
                1001 1110 = '158'
                1111 1111 = '255'

        Decimal equivalent of (1001 1110) is calculated as following (x^y represents the x to the power y):

        (1*2^7 + 0*2^6 + 0*2^5 + 1*2^4 + 1*2^3 + 1*2^2 + 1*2^1 + 0*2^0)
        = 128 + 0 + 0 + 16 + 8 + 4 + 2 + 0 = 158

        A byte consists of 8 bits. The highest number a byte can keep track of is '255'.
        Anything higher would require additional byte(s) to keep track of its decimal value.

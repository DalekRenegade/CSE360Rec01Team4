Tutorial 1 - Simultaneous Equations

▣ What is an equation?
► It is a statement which asserts that the values of two mathematical expressions are equal (indicated by the sign =).

▣ What are simultaneous equations?
► They are the equations involving two or more unknowns that are to have the same values in each equation.

        Consider the following equations:
                x + y = 6
                x - y = 2

        For the values [x = 4, y = 2] , both of our equations yield the desired result.
        To solve such equations, we equate one of the unknowns in the other equation and evaluate it.
        This gives us one of the values which we can use to find the other variable by equating it any of the given equations.

▣ What if we have more than 2 variables?
► We can solve a simultaneous equation with any number of unknowns. All we need is that the number of equations be equal to the number of unknowns.

        Consider the following equations with 3 unknowns:
                x + y + z = 6
                x + y - z = 0
                x - y + z = 2
        On solving it the same way, we get the answer [x = 1, y = 2, z = 3]
